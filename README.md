# 🚀 Nexus AI - Modern AI SaaS Landing Page

A modern, responsive AI SaaS landing page built with **React**, **Vite**, and **Tailwind CSS**. It features a clean UI, interactive pricing plans, testimonials, feature highlights, and a fully responsive design.

## 🌐 Live Demo

🔗 https://nexus-ai-landing-page-mu.vercel.app/

## 📂 GitHub Repository

🔗 https://github.com/architsinghal0005/nexus-ai-landing-page

---

## ✨ Features

- 🎨 Modern and responsive UI
- ⚡ Built with React + Vite
- 🎯 Fully responsive design
- 🚀 Interactive Hero section
- 💡 AI Features showcase
- 💰 Dynamic Pricing section
  - Monthly / Annual billing toggle
  - Currency switcher (USD, EUR, INR)
  - Featured "Most Popular" plan
- ⭐ Testimonials section
- 📞 Call-to-Action section
- 📌 Footer with newsletter subscription
- 🔗 Smooth navigation between sections

---

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/architsinghal0005/nexus-ai-landing-page.git
```

### Navigate to the project

```bash
cd nexus-ai-landing-page
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
